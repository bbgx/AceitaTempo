Privacidade por padrão

As preferências ficam no navegador e a extensão consulta só o necessário para calcular os tempos.

Como funciona

Três passos para transformar preço em tempo.

Você configura seu salário mensal e horas trabalhadas por mês, e a extensão faz o resto.

1

Defina seu perfil

Informe salário, moeda e horas mensais para calcular seu valor-hora.

2

Navegue normalmente

Abra lojas e páginas de produto. A extensão identifica os preços suportados.

3

Leia o tempo

Os preços ganham um badge discreto com a quantidade de horas ou minutos equivalentes.

Privacidade

Transparente sobre dados e comportamento.

A landing page aponta para a política oficial e para o canal de suporte da extensão.

  • As preferências do usuário ficam salvas no navegador via chrome.storage.
  • A extensão lê páginas para identificar preços e calcular o tempo de trabalho correspondente.
  • Não existe backend próprio para coleta ou compartilhamento comercial de dados.
  • O layout desta página é estático e compatível com GitHub Pages.
