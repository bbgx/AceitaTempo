Suporte

Precisa de ajuda com o AceitaTempo?

A forma mais rápida de falar com o projeto é abrir uma issue no GitHub. Se você encontrou um site que não está sendo detectado, envie a URL, um print e, se possível, o HTML do trecho do preço.

Por que não aparece em um site específico?

Alguns sites têm DOMs dinâmicos ou variações de layout. O recurso pode depender de heurística, então um print do trecho do preço ajuda bastante.

Como desativo em um site?

Abra as opções, deixe o switch de sites externos do jeito desejado e desmarque o site na lista de desativação manual.

Onde ficam as configurações?

As preferências ficam salvas no Chrome Storage do navegador. Elas podem sincronizar entre aparelhos, dependendo da configuração da conta Chrome.
